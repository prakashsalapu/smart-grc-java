# SmartGRC Java

Enterprise Governance, Risk & Compliance (GRC) Management System built using Core Java.

## Features

- Add model.Employee
- Display Employees
- Search model.Employee
- Update model.Employee
- Delete model.Employee

## Concepts Used

- Object-Oriented Programming
- Encapsulation
- Constructors
- Collections (ArrayList)
- CRUD Operations
- Method Overriding
- Validation
- Clean Code Principles

## Tech Stack

- Java
- IntelliJ IDEA
- Git
- GitHub
